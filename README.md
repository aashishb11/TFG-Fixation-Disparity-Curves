# TFG
This TFG will be done in two phases. 

Phase 1: An web application shall be created for demostrating Fixation disparity curves with the following rules : 

- we need to make one graph that shows 4 visual disparity curves each with different color for better comparision.
- Then we need to design a mathematic model to classify which type it belongs to (type 1 type 2 type 3 or type 4)
- To classify it in which type it belongs to we will approximate it on the basis of polynomical function.
- The inputs (x) of graph is fixed and (y) is the clinical parameter i.e the answer of patients.
- In total there are 7 parameters for each x and y.
- For each function there will be an error, we check the error.
- So we need to have a properly functioning webpage that shows 4 errores one for each type of curve and one graph with those 4 functions and 7 parameters.
- We put the errors below that are generated for each function.
- The calcualtion of errors in mentioned in the documentation 
https://pmc.ncbi.nlm.nih.gov/articles/PMC12682111/pdf/OPO-45-1642.pdf
